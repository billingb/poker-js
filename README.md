# poker-js

An javascript utility to identify the best poker hand from a 5 card hand.

## Running the script
* `npm run exec`

## Working on the project

### Prerequisites
*  node.js/npm

### Workspace Setup
*  Run `npm install` to pull down the project packages

### Linting
* Run `npm run lint`

### Unit Testing
* Run `npm test` or `npm test:watch`
